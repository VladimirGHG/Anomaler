# Anomaler

## Overview
Anomaler is a C++ system for streaming structured data to an anomaly detection model. It supports both training and real-time evaluation, enabling efficient end-to-end anomaly detection across multiple data types. The system is designed to be modular, flexible, and compatible with ML pipelines.

## Features
- **Generic Data Handling:** Accepts various structured data types using C++ templates.
- **Streaming Data:** Seamlessly feeds data to the anomaly detection model in real time.
- **Training & Evaluation:** Supports both model training and real-time anomaly detection.
- **Concurrent DataStreams:** Multiple streams can run independently, simulating real-world data pipelines.
